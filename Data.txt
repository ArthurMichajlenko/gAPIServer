// Курьер
ID              - уникальный код курьера
IMEI            - IMEI уникальный идентификатор ус-ва курьера
Tel             - номер телефона курьера
Name            - имя курьера

// Контрагенты
ID              - уникальный код клиента
Name            - имя клиента
Tel             - номер телефон клиента
Address         - адрес доставки

// Заказ
ID              - уникальный код заказа
CourierID       - код курьера исполняющего заказ (см. Курьер ID)
ClientID        - код контрагента кому предназначен заказ (см. Контрагенты ID)
PaymentMethod   - способ оплаты заказа (наличка, перечисление ...)
ConsistsTo[
    Product     - наименование товара
    Quantity    - количество
    Price       - цена
]    - перечень товара, который необходимо доставить клиенту
ConsistsFrom[
    Product     - наименование товара
    Quantity    - количество
    Price       - цена
]    - перечень товара, который необходимо забрать у клиента
OrderCost       - общая стоимость заказа
